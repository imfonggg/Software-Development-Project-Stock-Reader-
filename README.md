# Software-Development-Project-Stock-Reader-

Semester long project that showed how to use the .NET framework in visual studio. Using Windows Form, I made a stock reader program that utilizes the 4 pillars of Object Oriented Programming- Encapsulation, Abstraction, Inheritence, and Polymorphism. 

The stock reader takes input from an excel worksheet with stock data from various companies. The data is then sorted by date and using its open and close values to transform each data point into a candlestick for users to view. Candle sticks can also be shown by either days, weeks, or months.

Users can also change the period of time they wish the candlesticks to be shown, not showing those that are not in the relevant time frame using some basic sorting algorithms.

Finally, we added a simulate button where each candlestick renders in order and shows what type of candlestick it is, such as hammer, marubozu, etc.
